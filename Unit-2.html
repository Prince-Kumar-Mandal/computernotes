Unit-2:C Programming

1. Working with a Function

a. Define Function

A function is a self-contained block of code that performs a specific task when called.

b. Syntax of Functions

return_type function_name(parameters) {
    // body
}

c. Types of Functions

  • Library Functions: Predefined in C++ (e.g., sqrt(), strlen())
  • User-defined Functions: Created by the programmer

d. Components of Function

  1. Function Prototype: Declares the function before use
  2. Function Call: Invokes the function
  3. Function Definition: Contains the actual implementation
  4. Return Type: Specifies the data type of value returned

2. Categories of Functions with Examples

i. Return Type, No Arguments

int getNumber() {
  return 100;
}

ii. Return Type with Arguments

int add(int a, int b) {
  return a + b;
}

iii. No Return Type, No Arguments

void greet() {
  cout << "Hello!";
}

iv. No Return Type with Arguments

void display(int x) {
  cout << x;
}

3. Storage Classes

  • Automatic (auto): Default for local variables
  • External (extern): Declares a global variable used across files
  • Register: Hints to store variable in CPU register
  • Static: Retains value across function calls

4. Recursive Function

A recursive function calls itself until a stopping condition is met.

int factorial(int n) {
  if(n <= 1)
    return 1;
  else
    return n * factorial(n - 1);
}

5. Passing Array to a Function

Arrays can be passed as parameters to functions.

void printArray(int arr[], int size) {
  for(int i = 0; i < size; i++)
    cout << arr[i] << \" \";
}
